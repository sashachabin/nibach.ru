# nibach.ru

**@sashachabin** personal website **[nibach.ru](https://nibach.ru)** 

## Development

1. Install [Node.js](https://nodejs.org/en/download/) and [pnpm](https://www.npmjs.com/package/pnpm#user-content-installation)

2. Install dependencies
```
pnpm i
```

3. Run local server
```
pnpm start
```
